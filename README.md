# Price Prediction for Online Marketplace Products

## Overview
Online marketplaces host a wide variety of products with varying prices across platforms.
This project aims to build a machine learning model that predicts the price of a product
based on attributes such as brand, category, and other features.

**This project was developed using Google Colab for ease of collaboration and GPU support.**

## Dataset
The dataset used in this project was sourced from Kaggle and contains product
details such as name, brand, category, and price.

## Methodology
- Data Cleaning
- Exploratory Data Analysis (EDA)
- Model Training
- Model Evaluation

## Models Used
- Linear Regression
- Decision Tree Regressor

## Evaluation Metrics
- MAE
- RMSE
